pub fn from_bounds(
    eta: Eta,
    pb: &PartitionBound,
    x: &Vec<i64>
) -> Result<WeightTable, &'static str>[src]

Computes the weight table for the given bounds, enforcing an exact bound if exact total specified in exact.

Arguments

  • eta: privacy parameter
  • pb: a set of upper and lower bounds for the partition
  • x: the target (private) integer partition

Returns

A weight table consisting of the cumulative weights for all completions beginning with value q at index i.

Exact Arithmetic

This function calls enter_exact_scope() and exit_exact_scope(), and therefore clears the mpfr::flags and does not preserve the incoming flag state.

Known Timing Channels

This method does not compute worst-case arithmetic up front, and instead increases precision as needed during computation.

Computes the bias of sampling from the weight table. Stores helper tables of coefficients, total weights, etc, for later use if needed.

Arguments

  • pb: the PartitionBound to be used for sampling
  • x: the target partition x. (Note that this does not have to be the same target partition that was used to create the table.)

Returns

A vector indicating the bias of each index if x were the target partition.
